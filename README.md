📊 Sales Performance Dashboard – Power BI
📌 Project Overview

This project is an interactive Sales Performance Dashboard developed using Power BI to analyze business sales data across regions, states, categories, and customers.

The dashboard provides meaningful insights into:

Total Sales Performance

Profit Analysis

Customer Contribution

Shipping Mode Distribution

Category-wise & Sub-category-wise Profit

Monthly Profit Trends

📈 Key KPIs

💰 2M – Total Sales

📦 38K – Total Quantity

📊 286K – Total Profit

🧮 609K – Sum of AOV

📊 Dashboard Visualizations
🔹 Sales Analysis

Top Sales by State (Bar Chart)

Monthly Profit Trend (Column Chart)

Sales by Customer Name

🔹 Category Insights

Quantity by Category (Donut Chart)

Profit by Sub-Category (Horizontal Bar Chart)

🔹 Shipping Insights

Quantity by Ship Mode (Donut Chart)

🔹 Interactive Filters

Region (Tile Slicer)

State (Dropdown Slicer)

Reset Button using Bookmarks

🛠 Tools & Skills Used

Power BI Desktop

Data Cleaning

Data Modeling

DAX (Basic Measures)

Interactive Dashboard Design

Slicer & Bookmark Functionality

🧠 Business Insights Generated

Office Supplies contributes highest quantity.

Standard Class shipping is most preferred.

Certain states generate significantly higher sales.

Profit fluctuates month-wise.

Sub-categories like Copiers & Phones generate high profit.

📂 Dataset Details

The dataset contains:

Order Information

Customer Details

Product Categories

Shipping Data

Sales, Profit, Quantity & Discount

Total Rows: ~10,000
Total Columns: 20+
